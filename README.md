## 📖 Project Overview
This project analyzes retail sales data using SQL and Python to extract meaningful business insights. It focuses on understanding sales performance, profitability, customer behavior, and product trends.

The project combines SQL queries with Python visualization to deliver a complete data analysis workflow.

---

## 🎯 Objectives
- Analyze total sales and profit  
- Identify top-performing regions  
- Evaluate product category profitability  
- Discover top products and customers  
- Analyze shipping mode performance  
- Study monthly sales trends  

---

## 🛠️ Tech Stack
- Python 🐍  
- Pandas  
- SQLite (SQL)  
- Matplotlib  
- Seaborn  

---

## 📂 Dataset Features
- Order Details (Order ID, Date, Ship Mode)  
- Customer Information  
- Product Details (Category, Sub-category)  
- Sales, Quantity, Discount, Profit  
- Region and Location Data  

---

## 🔧 Data Preprocessing
- Removed duplicate records  
- Converted date columns into datetime format  
- Extracted month from order date  
- Loaded dataset into SQLite database  

---

## 🗄️ SQL Analysis Performed
- Total Sales Calculation  
- Total Profit Calculation  
- Sales by Region  
- Profit by Category  
- Top 10 Products by Sales  
- Top 10 Customers by Sales  
- Sales by Shipping Mode  
- Monthly Sales Trends  

---

## 📊 Key Insights
- West region generates highest sales  
- Technology category is most profitable  
- Few products contribute major revenue (Pareto effect)  
- Standard Class shipping dominates sales  
- Sales peak during November and December
